# デザイン方針

## 採用テーマ：パステル（D案）

`mockups/D-pastel.html` を参考実装とする。

## カラーパレット

| 変数 | 値 | 用途 |
|---|---|---|
| `--white` | `#ffffff` | カード・ウィジェット背景 |
| `--bg` | `#fafafa` | ページ背景 |
| `--text` | `#2c2c2c` | 本文テキスト |
| `--text-muted` | `#888888` | サブテキスト・日付など |
| `--mint` | `#b8f0d8` | パステル1（BR・メインアクセント） |
| `--mint-deep` | `#3dba7a` | ミント濃色 |
| `--sky` | `#b8e4f9` | パステル2（オーフェン系） |
| `--sky-deep` | `#3a9fd6` | スカイ濃色 |
| `--lavender` | `#dbbef9` | パステル3（大藪系） |
| `--lavender-deep` | `#9b59e8` | ラベンダー濃色 |
| `--peach` | `#ffd6b8` | パステル4（プロフィール系） |
| `--peach-deep` | `#e8834a` | ピーチ濃色 |
| `--yellow` | `#fff0a0` | お知らせ・強調 |
| `--yellow-deep` | `#c8a800` | イエロー濃色 |
| `--border` | `#e8e8e8` | 罫線・区切り |

## デザイン原則

- 白ベース、パステル多色でコンテンツを色分け
- カードにカテゴリごとの色を割り当て（バッジ＋トップバー）
- 影は控えめ（ふわっと感を維持）
- ナビはアンダーラインスタイル

## 比較検討したモック

| 案 | 方向性 | 結果 |
|---|---|---|
| A | Modern Green（緑系継承・クリーン） | 殺風景すぎ |
| B | Punk（黒背景・原色ネオン） | ダークテーマになった・保留 |
| C | Retro Modern（クリームベース懐古風） | 古い・ナシ |
| **D** | **Pastel（白ベース・パステル多色）** | **採用** |
